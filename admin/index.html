Workflow

GitHub maintenance

  1. Update content here.
  2. Export `site.json`.
  3. Commit the downloaded file into `data/site.json`.
  4. Commit any new portrait or gallery image files under `assets/images/`.
  5. Push to GitHub Pages.

Tip

Keep the gallery quiet

The gallery only opens when the URL path matches your secret path, for example `/nocturne-room`. It is intentionally absent from the main navigation.
